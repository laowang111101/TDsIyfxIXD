# 图书馆管理系统 python161

## 项目概述

图书馆管理系统是基于 Python 语言开发，使用 Flask 框架进行搭建的Web应用程序。该系统致力于提供一套高效、实用的图书馆管理解决方案。

## 技术栈

- **后端语言**: Python
- **Web框架**: Flask
- **数据库**: MySQL
- **开发工具**: PyCharm

## 功能模块

- 图书信息管理
- 读者信息管理
- 借阅信息管理
- 系统管理

## 系统角色

- 系统管理员
- 图书管理员
- 读者

## 运行环境

- 操作系统:不限
- Python版本: 3.x
- MySQL版本: 5.x 或更高

## 部署步骤

1. 安装 Python 环境。
2. 安装 Flask 框架。
3. 安装 MySQL 数据库，并导入数据脚本。
4. 将代码部署到服务器。
5. 运行应用程序。

## 目录结构

```
/library_system
|-- /app
|   |-- __init__.py
|   |-- models.py
|   |-- views.py
|-- /static
|-- /templates
|-- run.py
|-- requirements.txt
|-- database_script.sql
```

## 核心亮点

- 基于成熟稳定的 Flask 框架，易于维护和扩展。
- 使用 MySQL 作为数据库，保证数据稳定性和安全性。
- 完善的功能模块设计，满足日常图书馆管理需求。
